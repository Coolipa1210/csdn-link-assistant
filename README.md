# CSDN Link Assistant

CSDN 好友文章链接整理助手，用于从 CSDN 页面和白名单私信对话中收集文章链接，按状态整理、去重、导入导出，并辅助维护白名单待办流程。

所有数据默认保存在浏览器本地。本扩展不读取、不保存 CSDN 密码、Cookie 或 Token，也不是 CSDN 官方工具。使用时请遵守 CSDN 平台规则，并优先使用人工确认流程。

## 安装

1. 打开 Chrome 或 Edge 的扩展管理页面。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选择本项目目录。

## 功能

- 在 CSDN 页面扫描文章链接，自动去重。
- 按最近天数扫描白名单私信对话中的文章链接。
- 配置“我的作者 ID”，自动忽略自己的文章。
- 将待处理链接复制为 JS 数组或 JSON。
- 导入、导出白名单用户 ID，支持 TXT 一行一个 ID。
- 导入白名单用户 ID、私信链接或主页 URL，并按分组折叠管理。
- 白名单支持新建分组、展开分组用户、按用户或按整组获取 iframe。
- 选择分组和固定文本后，预览收件人并启动白名单处理队列。
- 支持审核模式：只填入内容，由用户手动确认后发送。
- 插件弹窗点击网页会关闭，这是浏览器限制；可点击“持久侧边栏”打开不会被网页点击关闭的面板。
- 插件会记住上次打开的功能页，下次打开自动回到同一页。
- 本地记录任务日志，不保存账号凭证。

## 打包

发布浏览器扩展时，将以下文件放在 zip 根目录：

- `manifest.json`
- `popup.html`
- `popup.css`
- `popup.js`
- `background.js`
- `content.js`
- `core.js`
- `assets/`
